# Calculadora

Um pequeno projeto de exemplo com operações matemáticas básicas, usado para praticar o fluxo de Pull Requests no GitHub.

## Uso

```python
from calculadora import somar, subtrair, multiplicar, dividir

somar(2, 3)       # 5
subtrair(5, 2)    # 3
multiplicar(2, 3) # 6
dividir(6, 3)     # 2
```
